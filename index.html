grads色テーブル (gradsスクリプト)

16番から80番の色番号に下記の色を対応させています。

  • 黒〜赤〜黄
    		•  [ hot.gs ] [ 色見本 ] (暖色系)
  • 水色〜紫
    		•  [ cool.gs ] [ 色見本 ] (寒色系)
  • 7色(虹色)
    		•  [ rgb.gs ] [ 色見本 ]
  • 青〜白〜赤
    		•  [ polar.gs ] [ 色見本 ] (負を青、正を赤など)
  • 青〜黒〜赤
    		•  [ split.gs ] [ 色見本 ] (負を青、正を赤で0付近を黒に)
  • 青〜白〜黄〜赤
    		•  [ jet.gs ] [ 色見本 ] (負を寒色系、正を暖色系など)
  • 茶〜水色〜青
    		•  [ drywet.gs ] [ 色見本 ] (乾燥〜湿潤の表現)
  • 青〜水色〜橙〜赤〜白
    		•  [ haxby.gs ] [ 色見本 ] (地形用?)
  • 赤〜白〜緑
    		•  [ red2green.gs ] [ 色見本 ]
  • 黒〜茶色〜薄茶色
    		•  [ copper.gs ] [ 色見本 ] (茶色系)
  • 黒〜青〜黄緑〜水色
    		•  [ ocean.gs ] [ 色見本 ] (海洋の色?)
  • 黒〜青〜茶〜肌色
    		•  [ relief.gs ] [ 色見本 ] (地形用?)
  • 赤〜橙〜黄〜緑〜青
    		•  [ seis.gs ] [ 色見本 ]
  • 赤〜黒(16〜70番のみ)
    		•  [ r2b.gs ] [ 色見本 ]
  • 灰〜黒(48〜80番のみ)
    		•  [ gray.gs ] [ 色見本 ] (グレースケールの図の作成)

    使用例:(gradsスクリプト中)

    'run /スクリプトを置いたディレクトリ/cool.gs' (色番号と色の対応を読み込み)
'set clevs -30 -20 -10  10  20  30' (データの値)
'set ccols  20  30  40  50  60  70  80' (値に対応する色番号)
'd 変数名'
    * これらの色見本に用いたRGBの配分はGMTに付属のmakecptを用いて計算しました。
